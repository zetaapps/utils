## Utils
Common utils methods 

### Install

Add jitpack to your root `build.gradle`
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```
	
Add the dependency
```
	dependencies {
	        compile 'com.github.zetaapps:utils:0.2'
	}
```
